# Fundamentos de Python

> Ejercicios básicos y ejemplos prácticos para aprender los fundamentos de Python.

## 📌 Descripción

Este proyecto es una colección de ejemplos y notas para quienes están iniciando en Python. El archivo principal `fundamentos.py` contiene conceptos básicos como variables, operadores, estructuras de datos y control de flujo.

## 🚀 Contenido del proyecto

- `fundamentos.py`: archivo principal con ejemplos comentados de:
  - Variables y nombres
  - Operadores matemáticos
  - Operadores de comparación
  - Condicionales anidados y lógicos
  - Estructuras de datos: listas, tuplas y diccionarios
  - Bucles `for`
  - Ejemplos prácticos y comentarios de apoyo
- `fumdamentos_python.txt`: notas resumidas sobre los temas de Python tratados.

## 🧠 Temas cubiertos

1. Variables y buenas prácticas de nombres
2. Operadores matemáticos
3. Comparaciones y expresiones booleanas
4. Condicionales `if`, `elif` y `else`
5. Bucles `for`
6. Listas y manipulación de índices
7. Diccionarios y acceso a datos anidados
8. Uso de `print` y formato de cadenas

## ✅ Ejemplo de uso

Para ejecutar el ejemplo principal del proyecto, abre una terminal en la carpeta del proyecto y ejecuta:

```bash
python fundamentos.py
```

Este script recorre una lista de nombres y muestra un saludo personalizado en la consola.

## 📚 Qué aprenderás

- Cómo definir y usar variables en Python
- Cómo sumar, restar, multiplicar y dividir valores
- Cómo comparar valores con operadores relacionales
- Cómo usar condicionales para controlar el flujo del programa
- Cómo trabajar con listas y diccionarios
- Cómo iterar elementos con bucles `for`

## 💡 Notas

- El código contiene varios bloques comentados que sirven como ejemplo de uso y explicaciones.
- Los comentarios en `fundamentos.py` ayudan a comprender cada concepto paso a paso.

## 🛠️ Recomendaciones

- Abre `fundamentos.py` y descomenta bloques pequeños para practicar.
- Modifica valores y observa cómo cambian los resultados.
- Añade nuevos ejemplos propios para reforzar el aprendizaje.
